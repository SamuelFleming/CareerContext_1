# CareerContext — Portfolio Story

## Developer Context

CareerContext comes from a real job-market problem faced by the developer.

The developer is building a career profile across:

- full-stack development experience
- public sector software work
- university software projects
- technical leadership experience
- MERN stack development
- .NET, Angular, SQL, and API development
- emerging AI-first development practices
- personal software projects and portfolio work

The developer already uses AI tools to evaluate jobs and draft cover letters. However, the repeated difficulty is maintaining a reliable body of personal career context that can be reused across different applications.

## The Problem Behind the Project

AI can write strong application material when it has good context.

The pain point is not simply writing the cover letter. The pain point is:

- remembering the right experiences
- retrieving the strongest examples
- keeping project details organised
- turning rough notes into polished summaries
- matching experience evidence to job requirements
- avoiding generic, unsupported AI-generated claims
- keeping application content grounded in real work

CareerContext is designed to solve that context-management problem.

## Portfolio Story

CareerContext is an AI-first MERN stack application that demonstrates how large language models can be integrated into a practical, domain-specific workflow.

The project shows the developer's ability to:

- model a real-world domain
- design a useful MVP from personal workflow pain
- build a full-stack MERN application
- implement AI-assisted user flows
- structure unstructured data into reusable application context
- use AI for extraction, summarisation, retrieval, evaluation, and document generation
- design a human-in-the-loop system where generated content can be reviewed and edited
- think about UX, information architecture, and user productivity
- build software that supports the developer's own job search while also acting as a portfolio artefact

## Why This Project Is Interesting

CareerContext is intentionally meta.

It is a job-market portfolio project that directly helps the developer apply for jobs. The product itself becomes part of the developer's professional story:

> I built an AI-first application to solve a problem in my own job search: converting scattered career experience into a structured, reusable, AI-assisted career evidence system.

## Employer-Relevant Skills Demonstrated

The project can demonstrate:

- MERN stack application development
- React component design

- AI integration

## Project Positioning Statement

CareerContext is not just a cover letter generator.

It is a full-stack AI-assisted career context platform that helps a user organise their work history, reason about job fit, and generate application material grounded in stored experience evidence.
