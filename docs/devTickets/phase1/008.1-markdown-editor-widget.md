## Status

**Implemented**

## Phase

Phase 1 — Auth and Profile (reused in later phases)

Future reuse: Experience, Activity, Opportunity, Document, AI outputs.

## Objective

Create a reusable native Markdown editor widget for CareerContext raw and polished text inputs — Edit/Preview modes with optional formatting toolbar.


## Acceptance Criteria

- [x] `MarkdownEditor` component exists and is controlled via `value` / `onChange`
- [x] Edit and Preview modes with non-editable preview rendering
- [x] Profile raw summary uses `MarkdownEditor`
- [x] Journal drawer notes use the same `MarkdownEditor` component
- [x] Styling matches Dashboard/Profile visual direction
- [x] Empty preview state handled gracefully
- [x] Formatting toolbar implemented (bold, italic, lists, heading, code)
- [x] Component reusable outside Profile

## Out of Scope

- Full WYSIWYG, image upload, file import/export, collaborative editing, AI inline editing, version history
