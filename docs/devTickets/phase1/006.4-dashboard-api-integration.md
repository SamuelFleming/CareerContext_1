### Component mapping

| API key | Component | Notes |
|---------|-----------|-------|
| `identity` + `interactiveCv` | `InteractiveCvCard` | CV header, summary, skill chips, evidence cards |
| `profileCompleteness` | `ProfileCompletenessPrompt` | Render only when `showPrompt === true`; shows `completed/total (score%)` |
| `evidencePanel` | `EvidencePanel` | Collapsible toggle: Evidence Summary ↔ Recent Activity |

### Removed from Dashboard UI

- `CoreResumePreviewCard` (deleted)
- `QuickActionsCard` (deleted)

### Phase 1 mock fallback

`InteractiveCvCard` uses `resolvePhase1InteractiveCv()` to display mock chips and evidence cards when the API returns empty arrays (e.g. stale server process).
