# CareerContext — The Idea

> an AI-assisted career evidence system that helps users capture, organise, retrieve, and reuse their professional, academic, and project experience to evaluate job opportunities and generate tailored application documents.

## Problem

Job applications require more than a resume. Strong applications need tailored evidence: relevant experience, project details, technologies used, responsibilities, outcomes, and a clear career narrative.

The problem is that this information is usually scattered across:

- resumes
- rough work notes
- university project documents
- GitHub repositories
- cover letter drafts
- LLM conversations
- job descriptions
- your memory

This makes the process of completing a job application slow. Even when AI tools can write good cover letters, the difficult part is consistently providing them with accurate, relevant, reusable context.

## Product Idea

CareerContext is an AI-assisted career evidence and job application workspace that helps users store, organise, retrieve, and reuse their professional, academic, and project experience.

It acts as a persistent personal career context layer that can evaluate job opportunities and generate grounded application documents such as cover letters and targeted resume content.

**Evolutionary Thinking**: Positive app outcomes (such as an in-depth professional experiecn profile, accurate opportunity evaluations and documentation derivation) relies on conistency and quality of the user input. Hence user engagement is a necessity for an evolutionary product - target: CareerContext will take on the usage as a 'Professional Journal'. With a design focus on an engaging UI and effcetive outputs is ess4ential for the app to evolve into a daily tool for users to document notes/summaries of the work they do whether in the hunt for new work or in a position they already love - [see more here](./03_solution_technology.md#ux-direction).

## Elevator Pitch

>**CareerContext** helps job seekers turn messy career experience into a structured, AI-searchable evidence base that can be used to evaluate jobs, compare opportunities, and generate tailored application documents.

## High-Level Scope

The MVP focuses on four core capabilities:

1. **Profile and Core Context**
   - Store basic user profile information.
   - Store a user-written career context summary.
   - Upload or paste a core resume.

2. **Experience and Activity Capture**
   - Create experiences such as jobs, projects, courses, and certifications.
   - Add activities, achievements, responsibilities, notes, and technologies.
   - Support rough journal-style entries that can later be converted into structured activities.

3. **Opportunity Management**
   - Add job opportunities by pasting job descriptions.
   - Store role, company, origin/source, and opportunity notes.
   - Use AI to extract requirements, skills, responsibilities, and role signals.

4. **AI-Assisted Evaluation and Documents**
   - Evaluate user fit against an opportunity.
   - Identify relevant experiences and activities.
   - Compare multiple opportunities.
   - Generate cover letter drafts and targeted resume content.

## Out of Scope for Initial MVP

The first MVP will not focus on:

- production deployment
- job board scraping
- payment systems
- complex document formatting
- multi-user collaboration
- browser extensions
- full applicant tracking workflows
- advanced analytics dashboards
- highly polished wireframes before development starts

## Product Principle

CareerContext should not be a generic AI text generator.

It should be a **career evidence system** where AI is used to clean, structure, retrieve, evaluate, and generate content from a user's actual experience.
